🔧 Technologies Clés

🖥️ Hardware

  • Raspberry Pi Zero 2 → ESP32-C3
  • ELM327 (Puce PIC18F25K80)
  • TFT LCD 1.28" Waveshare
  • ESP32-S3 Zero + GC9A01
  • TTP223 Touch Sensor
  • Modules Audio (MAX98357A/INMP441)

💻 Software

  • Python 3 + PIL
  • Serial OBD (38400 8N1)
  • C++/Arduino + TFT_eSPI
  • ISO 14230-4 KWP Protocol
  • VW Handshake Sequence
  • OBD2 PIDs (010C, 010D, 0105)

🤖 IA / Assistant

  • OpenClaw Assistant Maxi
  • Multi-modèles (GLM-4.7, Gemini)
  • Gestion des tâches
  • OpenClaw Agent
