Ce document est la documentation officielle de la bibliothèque `python-OBD`, une ressource logicielle pour l'interfaçage avec le port OBD-II des véhicules. La bibliothèque permet de récupérer des données de capteurs en temps réel et de réaliser des diagnostics. Elle est particulièrement adaptée pour des projets sur Raspberry Pi et fonctionne avec les adaptateurs ELM327. La documentation détaille l'installation via `pip`, l'utilisation des commandes de base (e.g., `obd.commands.SPEED`), et la structure modulaire de la bibliothèque. Elle est cruciale pour le développement de la partie logicielle d'It bobd en Python, notamment pour l'extraction des données véhicule.
