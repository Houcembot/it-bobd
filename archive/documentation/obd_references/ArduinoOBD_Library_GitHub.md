Ce dépôt GitHub contient la bibliothèque ArduinoOBD, une solution open-source pour interfacer un Arduino avec le port OBD-II des véhicules. En plus de l'accès facile aux données OBD-II, elle intègre un capteur de mouvement 9-DOF et un voltmètre. Le projet liste les PIDs (Parameter IDs) les plus utilisés (RPM, charge moteur, température, vitesse, etc.) et fournit des exemples de code pour l'affichage des données sur différents types d'écrans (OLED, TFT). Cette bibliothèque est une ressource technique essentielle pour le développement logiciel de l'interface OBD-II d'It bobd.
