Ce document présente l'AutoPi Mini, un dispositif télématique "plug & play" conçu pour la gestion de flotte et la collecte de données via le port OBD-II. Il est optimisé pour les véhicules électriques (EV) et les moteurs à combustion interne (ICE), supportant un large éventail de paramètres OEM ainsi que les protocoles OBD legacy (K-Line, KWP). Le rapport détaille les capacités matérielles (4G/LTE intégré, conception compacte), la facilité d'installation, et les nombreux paramètres véhicules que l'appareil peut monitorer, ce qui inclut la charge, la santé et la température de la batterie EV, ainsi que des données standard comme le RPM et la vitesse pour les ICE. L'intégration avec AutoPi Cloud est également mise en avant.

Ce produit est une solution clé en main intéressante du point de vue concurrentiel pour It bobd, et la liste des paramètres supportés est une référence technique précieuse pour le développement.
