Ce tutoriel détaillé de Hackster.io explique comment transformer un Raspberry Pi en une "boîte noire" OBD-II pour véhicule. Il décrit la connexion du Raspberry Pi à un adaptateur Bluetooth ELM327 et l'affichage des données moteur en temps réel sur l'unité principale (autoradio) de la voiture. Le guide inclut la liste du matériel nécessaire, les étapes d'installation logicielle (utilisant la bibliothèque Python `pyOBD`) et les instructions pour l'intégration physique dans le véhicule. Il aborde également la journalisation des données.

Très pertinent pour l'intégration OBD et le développement logiciel Python pour It bobd.
