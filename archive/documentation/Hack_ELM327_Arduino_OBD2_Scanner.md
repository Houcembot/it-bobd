Ce document explique comment modifier un câble ELM327 OBD2 pour créer un scanner Arduino. La procédure inclut le contournement de la puce USB et la connexion directe des broches série à un Arduino équipé d'un écran LCD. Le document fournit également le code nécessaire pour lire des données véhicule comme la température, le régime moteur et la vitesse via le port OBD2.

Très pertinent pour la phase de développement hardware et logiciel d'It bobd.
