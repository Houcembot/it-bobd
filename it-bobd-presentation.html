Qu'est-ce que IT-BOBD ?

IT-BOBD est un robot embarqué qui se branche sur le port OBD2 de ta voiture. Il lit les données moteur en temps réel (RPM, vitesse, température, 40+ paramètres) et les exprime via des animations de yeux sur un écran TFT rond de 1.28". C'est à la fois un outil de diagnostic, un compagnon émotionnel et un projet DIY open-source.

Comment ça marche ?

Une connexion filaire directe, zéro Bluetooth, zéro lag.

🔌
Connexion OBD2
Un câble 5 fils (RX, TX, VCC, GND) relie l'ELM327 PIC18F25K80 directement au port OBD2 de la voiture. Protocole ISO 14230-4 KWP avec handshake VW.
⚡
ESP32-S3 Zero
Le microcontrôleur reçoit les données via UART Serial1 (38400 8N1), les décode, et décide quelle expression afficher selon le contexte de conduite.
👁️
Expressions Visuelles
L'écran GC9A01 TFT round 240×240 affiche les yeux animés. RPM élevé = colère, régime stable = calme, moteur froid = somnolence… et bien plus.
  OBD2 port voiture (VW Polo 9N)
       ↕  ISO 14230-4 KWP
  ELM327 PIC18F25K80
       ↕  UART (TX=GPIO8 / RX=GPIO9 / 38400 8N1)
  ESP32-S3 Zero
       ↕  SPI
  GC9A01 Round TFT 1.28" (240×240)
       → Animations yeux (NORMAL / ANGRY / HAPPY / TIRED)

Historique du projet

Premiers tests OBD2 — Échecs
Connexion Bluetooth ELM327 standard — impossible de lire les données VW. Diagnostic : protocole VAG non standard.
15 Jan 2026 🎉
BREAKTHROUGH — Handshake VW
Découverte de la séquence AT SH 81 10 F1. Première lecture RPM stable. Le projet décolle.
Jan → Fév 2026
Dashboard LCD, Coach Trip, Tactile, Migration ESP32
Développement du dashboard Python (Raspberry Pi), animations yeux, migration vers ESP32-S3 Zero + GC9A01.
